# ENSEA Tiny Shell

This project implements a basic command-line shell program named *enseash*. The shell is designed to execute commands, display return codes, manage execution times, handle redirections, and support execution of complex commands.

## Features

1. *Welcome Message & Prompt*: 
   - Displays a welcome message on startup.
   - The prompt (enseash %) is displayed for user input.

2. *Execution of Commands*: 
   - Executes commands entered by the user.
   - Returns the exit status or signal of the previous command.

3. *Command Execution Time*:
   - Measures and displays the execution time of each command.

4. *Handling of Complex Commands*:
   - Supports commands with arguments (e.g., hostname -i).

5. *Redirection Management*:
   - Supports standard input (<) and output (>) redirection for commands.
